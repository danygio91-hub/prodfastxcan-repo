# **App Name**: ProdTime Tracker

## Core Features:

- Secure Login: Secure login using operator name and password.
- Dashboard: Dashboard with prominent buttons for core functions.
- Operator Data: Button to view and manage operator data.
- Scan Job Order: Button to scan barcode the job order (commessa).
- Problem Reporting: Button to report any issue during production.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) for a clean and neutral background.
- Accent color: Orange (#FF9800) to highlight key interactive elements such as buttons.
- Body and headline font: 'PT Sans' (sans-serif) for a modern, readable style that balances clarity with a touch of warmth.
- Use clear, easily recognizable icons from a standard library like Material Icons to represent each function.
- Dashboard should be simple and intuitive, using a grid-based layout for easy navigation.
- Subtle animations to indicate a successful scan, data entry, or problem submission.

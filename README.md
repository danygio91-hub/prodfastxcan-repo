# ProdFast Xcan

Questo è uno starter NextJS in Firebase Studio.

Per iniziare, dai un'occhiata a src/app/page.tsx.

## Deployment

Questa applicazione è configurata per il deployment su **Firebase App Hosting**, una piattaforma moderna e ottimizzata per applicazioni web come Next.js. La configurazione si trova nel file `apphosting.yaml`.

Firebase App Hosting gestisce automaticamente la build e il rilascio della tua applicazione, rendendola accessibile online tramite un URL pubblico (es. `https://tuo-progetto.web.app`). Non è necessario configurare la sezione "Firebase Hosting" (il servizio classico) dalla console di Firebase.
